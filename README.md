# 🤖 RAG ChatBot

セキュアな知識ベース検索システム

## 機能
- 🔐 JWT認証システム
- 🛡️ レート制限・CORS保護
- 💬 インテリジェントなRAG検索
- 📚 引用情報表示
- 🎨 美しいStreamlit UI

## アーキテクチャ
- Frontend: Streamlit Cloud
- Backend: AWS Lambda + API Gateway
- Database: DynamoDB
- Security: Gateway Response + Rate Limiting

## セキュリティ機能
- ✅ HTTPメソッド制限
- ✅ CORS保護
- ✅ JWT認証
- ✅ レート制限
- ✅ 攻撃パターン検出

## 使い方
1. サインアップでアカウント作成
2. ログインしてチャット開始
3. AI powered な回答
